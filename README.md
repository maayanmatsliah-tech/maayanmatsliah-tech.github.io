# Maayan Matsliah — Personal Portfolio

🌐 **Live site:** [maayanmatsliah-tech.github.io](https://maayanmatsliah-tech.github.io)

A personal portfolio website built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

## About

This is the source code for my personal portfolio site. It showcases my projects, skills, and contact information. The site is designed to be clean, fast, and recruiter-friendly.

---

## Features

- **Responsive design** — works on desktop and mobile
- **Split hero layout** — photo and intro side by side
- **Project cards** — with tech stack tags and GitHub links
- **Sticky navigation** — smooth scrolling to each section
- **No dependencies** — pure HTML, CSS, and JavaScript

---

## Tech Stack

- HTML5
- CSS3 (Flexbox, CSS Grid, custom properties)
- JavaScript (vanilla)
- Hosted on GitHub Pages
